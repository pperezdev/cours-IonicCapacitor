En utilisant le plugin de notifications, créez une application simple permettant aux utilisateurs de définir des rappels quand ils le souhaitent.

L’interface doit être proche de celle de l’application « Horloge », avec une liste de toutes les notifications et la possibilité d’en ajouter une nouvelle ou d’en supprimer une existante.

⸻

En utilisant le plugin de préférences, vous devez stocker certaines informations de votre projet.

Créez une simple liste de tâches (TODO list) qui doit persister même après la fermeture de l’application.

Vous devez pouvoir :

* Créer un nouvel élément
* Modifier un élément existant
* Supprimer un élément
* Afficher le contenu complet d’un élément
